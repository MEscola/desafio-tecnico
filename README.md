# Sistema de GEstão de Projetos

## Stack
- Python
- Django
- SQLite

## Como rodar o projeto
1. Crie o ambiente virtual: `python -m venv venv`
2. Ative: `venv\Scripts\activate` (Windows)
3. Instale: `pip install -r requirements.txt`
4. Migrações: `python manage.py migrate`
5. Rode: `python manage.py runserver`
